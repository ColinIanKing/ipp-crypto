.. _sm3init:



SM3Init
=======


Initializes user-supplied memory as IppsSM3State context for future use.


Syntax
------


IppStatus ippsSM3Init(IppsSM3State\* pCtx);


Include Files
-------------


``ippcp.h``


Parameters
----------


.. list-table:: 
   :header-rows: 0

   * -     pCtx   
     -  Pointer to the IppsSM3State context being intialized.




Description
-----------


The function initializes the memory pointed by pCtx as IppsSM3State
context.


Return Values
-------------


.. list-table:: 
   :header-rows: 0

   * -     ippStsNoErr   
     -  Indicates no error. Any other value indicates an error or warning.
   * -     ippStsNullPtrErr   
     -  Indicates an error condition if any of the specified pointers is NULL.
