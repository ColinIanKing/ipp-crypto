What's New
==========


The document has been updated with the following changes to the product:


-  Added new sections to the Multi-buffer Functions chapter: `Edwards
   Curve25519 Elliptic Curve
   Functions <edwards-curve25519-elliptic-curve-functions.html>`__, `SM2
   Elliptic Curve
   Functions <sm2-elliptic-curve-functions.html>`__, `SM4
   Algorithm
   Functions <sm4-algorithm-functions.html>`__,
   `Modular
   Exponentiation <modular-exponentiation.html>`__.


-  Added the
   `GFpECInitStd <gfpecinitstd.html>`__,
   `GFpECPrivateKey <gfpecprivatekey-gfpecpublickey-gfpectstkeypair.html>`__,
   `GFpECPublicKey <gfpecpublickey.html>`__,
   `GFpECTstKeyPair <gfpectstkeypair.html>`__,
   `GFpECPSharedSecretDH,
   GFpECPSharedSecretDHC <gfpecpsharedsecretdh-gfpecpsharedsecretdhc.html>`__,
   `GFpECPSignDSA, GFpECPSignNR,
   GFpECPSignSM2 <gfpecpsigndsa-gfpecpsignnr-gfpecpsignsm2.html>`__,
   `GFpECPVerifyDSA, GFpECPVerifyNR,
   GFpECPVerifySM2 <gfpecpverifydsa-gfpecpverifynr-gfpecpverifysm2.html>`__
   functions.


-  Added the
   `AES_EncryptCFB16_MB <aes_encryptcfb16_mb.html>`__
   function.


-  Added information on `Security Validation of Library
   Functions <security-validation-of-library-functions.html>`__.


-

      Expanded the description for the
      `GFpECSetPoint <gfpecsetpoint-gfpecsetpointregular.html>`__,
      `GFpECSetPointHash <gfpecsetpointhash-gfpecsetpointhashbackcompatible-gfpecsetpointhash_rmf-gfpecsetpointhashbackcompatible_rmf.html>`__,
      and
      `GFpECGetPoint <gfpecgetpoint-gfpecgetpointregular.html>`__
      functions.


-  Added the section `Multi-buffer Cryptography
   Functions <multi-buffer-cryptography-functions.html>`__. Added
   the
   `mbx_rsa_public <mbx_rsa_public.html>`__,
   `mbx_rsa_private <mbx_rsa_private.html>`__,
   `mbx_rsa_private_crt <mbx_rsa_private_crt.html>`__,
   `mbx_RSA_Method_BufSize <mbx_rsa_method_bufsize.html>`__,
   `mbx_nistp256/384/521_ecdsa_sign_setup <mbx_nistp256-384-521_ecdsa_sign_setup.html>`__,
   `mbx_nistp256/384/521_ecdsa_sign_complete <mbx_nistp256-384-521_ecdsa_sign_complete.html>`__,
   `mbx_nistp256/384/521_ecdsa_sign <mbx_nistp256-384-521_ecdsa_sign.html>`__,
   `mbx_nistp256/384/521_ecdsa_verify <mbx_nistp256-384-521_ecdsa_verify.html>`__,
   `mbx_nistp256/384/521_ecpublic_key <mbx_nistp256-384-521_ecpublic_key.html>`__,
   `mbx_nistp256/384/521_ecdh <mbx_nistp256-384-521_ecdh.html>`__,
   `mbx_x25519_public_key <mbx_x25519_public_key.html>`__,
   `mbx_x25519 <mbx_x25519.html>`__
   functions.


-  Added
   `HashMethodSet <hashmethodset.html>`__,
   `HashMethodGetSize <hashmethodgetsize.html>`__
   functions.


-  Added `SM3 Hash
   Functions <sm3-hash-functions.html>`__.

- Added mitigation for Frequency Throttling Side-Channel Attack (Frequency Throttling Side Channel Software Guidance for Cryptography Implementations) for ECB, CMAC and GCM modes of AES. 

- Extended Crypto Multi-buffer library with CCM and GCM modes of SM4 algorithm.

- Fixed in-place mode of executing (when pa_out == pa_inp) for the SM4 CBC and CFB modes in Crypto Multi-buffer library.

- Added new API that updates pointer to ``IppsHashMethod`` context inside the ``IppsHashState_rmf`` state.

- Fixed ``ippsHMAC_Pack(_rmf)``, ``ippsHMAC_Unpack(_rmf)`` APIs. Now the context id is set up properly during the packing-unpacking process.

Additionally, minor updates have been made to fix inaccuracies in the
document.
