.. _sha1init:




SHA1Init
========


Initializes user-supplied memory as IppsSHA1State context for future
use.


Syntax
------


IppStatus ippsSHA1Init(IppsSHA1State\* pCtx);


Include Files
-------------


``ippcp.h``


Parameters
----------


.. list-table:: 
   :header-rows: 0

   * -     pCtx   
     -  Pointer to the IppsSHA1State context being intialized.




Description
-----------


The function initializes the memory pointed by pCtx as IppsSHA1State
context.


Return Values
-------------


.. list-table:: 
   :header-rows: 0

   * -     ippStsNoErr   
     -  Indicates no error. Any other value indicates an error or warning.
   * -     ippStsNullPtrErr   
     -  Indicates an error condition if any of the specified pointers is NULL.
