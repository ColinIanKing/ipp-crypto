.. _getenablednumthreads:


GetEnabledNumThreads
====================


Returns the number of existing threads in the multithreading
environment.


Syntax
------


int ippcpGetEnabledNumThreads(void);


Include Files
-------------


``ippcp.h``


Description
-----------


This function returns the number of the OpenMP\* threads specified by
the user. If the number is not specified, the function returns the
initial number of threads, which depends on the number of logical
processors.
