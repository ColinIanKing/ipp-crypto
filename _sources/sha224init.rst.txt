SHA224Init
==========


Initializes user-supplied memory as IppsSHA224State context for future
use.


Syntax
------


IppStatus ippsSHA224Init(IppsSHA224State\* pCtx);


Include Files
-------------


``ippcp.h``


Parameters
----------


.. list-table:: 
   :header-rows: 0

   * -     pCtx   
     -  Pointer to the IppsSHA224State context being intialized.




Description
-----------


The function initializes the memory pointed by pCtx as IppsSHA224State
context.


Return Values
-------------


.. list-table:: 
   :header-rows: 0

   * -     ippStsNoErr   
     -  Indicates no error. Any other value indicates an error or warning.
   * -     ippStsNullPtrErr   
     -  Indicates an error condition if any of the specified pointers is NULL.
