.. _sha224pack-sha224unpack:



SHA224Pack, SHA224Unpack
========================


Packs/unpacks the IppsSHA224State context into/from a user-defined
buffer.


Syntax
------


IppStatus ippsSHA224Pack (const IppsSHA224State\* pCtx, Ipp8u\*
pBuffer);


IppStatus ippsSHA224Unpack (const Ipp8u\* pBuffer, IppsSHA224State\*
pCtx);


Include Files
-------------


``ippcp.h``


Parameters
----------


.. list-table:: 
   :header-rows: 0

   * -     pCtx   
     -  Pointer to the IppsSHA224State context.
   * -     pBuffer   
     -  Pointer to the user-defined buffer.




Description
-----------


The SHA224Pack function transforms the \*pCtx context to a
position-independent form and stores it in the \*pBuffer buffer. The
SHA224Unpack function performs the inverse operation, that is,
transforms the contents of the \*pBuffer buffer into a normal
IppsSHA224State context. The SHA224Pack and SHA224Unpack functions
enable replacing the position-dependent IppsSHA224State context in the
memory.


Call the
`SHA224GetSize <sha224getsize.html>`__
function prior to SHA224Pack/SHA224Unpack to determine the size of the
buffer.


Return Values
-------------


.. list-table:: 
   :header-rows: 0

   * -     ippStsNoErr   
     -  Indicates no error. Any other value indicates an error or warning.
   * -     ippStsNullPtrErr   
     -  Indicates an error condition if any of the specified pointers is NULL.
